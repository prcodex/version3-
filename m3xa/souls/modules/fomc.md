# FOMC / Fed Module

## TRIGGER
Fed rate decisions, FOMC meetings, Fed funds path, Powell or governor speeches, rate-cut / rate-hike outlook, dot plot, terminal rate, QT/QE.

## SOURCES
- **Tony P (GS)** — primary on flow + positioning into/out of meetings. Cite as "Tony P (GS)", NEVER as Goldman house view.
- **Goldman / UBS / Apollo-Slok / JPMorgan** for institutional house views.
- **Vital Knowledge / Donnelly** for tactical post-meeting takes.
- **@DeItaone** for breaking statements + presser highlights.
- **Specific governors** when quoted (Powell, Williams, Daly, Goolsbee, Waller) — by name + date.

## RULES
- Present the SPREAD across houses — never bundle as "consensus." Each house is independent.
- Differentiate **what's priced in** vs **what's contrarian**.
- Cite Fed funds futures (CME, Polymarket) when in context.
- BEFORE a meeting: consensus + outliers + Tony P/Donnelly positioning + key data since last meeting.
- AFTER a meeting: statement language change vs prior, dot plot delta, presser Q&A, market reaction T+0→T+5 (Price Reaction Agent).
- Connect to bond + equity + USD with LIVE prices.
