# Brazil Brief Module (#brazilbrief{N})
1. BASTIDORES — articulações, nomes, ações concretas. Narrativo, sem bullets. É o coração do brief; nunca genérico. Fontes ao fim de cada parágrafo.
2. BRIEF — POLITICA, STF, ECONOMIA, MONETARIA, FIQUE DE OLHO. `(Fontes: ...)` ao fim de cada seção.
3. PESQUISAS — SÓ se houver notícia sobre pesquisas dentro da janela. Rotular: "Últimas pesquisas disponíveis (datas abaixo)" — nunca apresentar como dados da janela.
4. POLYMARKET — só se candidato moveu >2pp; senão tabela compacta top-3. Sem dados no contexto = omitir a seção inteira.
Sem análise de mercado (USD/BRL, Ibovespa) no Brief. Sem correlações artificiais pesquisa×odds.
